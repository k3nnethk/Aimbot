Aimbot.py is a program that can be used in all aiming and firing platforms. This can include games as well as other appications.
In order to impliment the code update the name of the appication being used from 'Ghost Recon Wildlands' to the new game or appication name. 
save changes and run the game or application you want to use.
Run the code and direct the game character towards enemy combatants and the code will automatically aim and fire at the targets.
The targeting model can be interchanged with a custom trained model for higher accuracy or for games with non-human targets.
